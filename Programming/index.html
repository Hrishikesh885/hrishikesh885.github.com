SWITCH STATEMENT:-

The switch statement is used to perform various tasks using cases depending upon value of a variable or expression. It is known as multiple branching statement or user choice statement.We use switch statement when we have different types of operations to be performed based on the choice given by the user.ITS A MENU DRIVEN PROGRAM. Some terms of switch:-

  • SWITCH - The switch statement executes the case corresponding to the value of the value of the expression/constant/choice selected.
  • CASE - The case keyword is follow by an integer constant or a character constant and a colon ( : ) symbol followed by one or more valid statements.
  • BREAK - The break statement causes the termination of execution of the particular statement(s) given after the closing brace of a switch.If there is no break,the execution flows through and executes all the cases till the control gets a break statement.This situation is known as 'FALL THROUGH'.
  • DEFAULT - Equivalent of else in if-else but for switch only.Written after the last case.Executes if none of the conditions are met.

Types of programming paradigms:-

The process of organizing principals of a program is known as programming paradigm.It is an approach to programming.

The following can be treated as programming paradigms:-

  • Procedural programming:- The procedural programming emphasis on doing things.A list of instructions written to perform a task where each statement instructs the computer to do some task.This is known as procedural programming.
  • Modularity programming:- To reduce the size of a long program,the program is divided into smaller groups and these groups can be termed as modules.So a set of related procedures that perform a task is known as module and the program designed using this concept is known as modular programming.
  • Object Oriented Programming:- The programming concept which is implemented using an object is known as object oriented programming.

What is an object?

An object is an identifiable entity with some characteristics and behaviours .

The following are the characteristics of an object:-

  • IDENTITY:- It is the name by which an object is recognised.
  • STATE:- The object may be in any state i.e. either processed or unprocessed.For example the computer may be in an in or off or out of order state.
  • BEHAVIOUR:- This refers that what an object is capable of doing.

Concepts of OOP:-

These concepts have been developed to overcome the drawbacks of conventional programming concept in order to make program easy and understandable. REMEMBER THEM AS ACRONYM- A.P.I.E (pronounce apie)

The following are the concepts of object oriented programming:-

  • ABSTRACTION:- The act of representing essential features of an object without knowing the background and details.
  • POLYMORPHISM:- It is the ability of representing a thing in more than one form.
  • INHERITANCE:- It is the property of inheriting the properties of one class from some other class. The class which inherits its own properties is known as Base/Super Class. The class which inherits another class is known as Derived/Sub Class.
  • ENCAPSULATION:- Wrapping up of data and its associated function into a single unit .

Everything related to tokens:-

Smallest individual units in a program are known as tokens. THE TYPES OF TOKENS CAN BE REMEMERED BY THE ACRONYM -- K.I.L.P.O

The following are the Tokens:-

  • KEYWORDS:- They hold a special place in a programming language as they represent a statement or function to perform a task. eg:- if,for,while,do,continue,case,etc.
  • IDENTIFIERS/VARIABLES:- Those quantities which change their values during the program execution are called identifiers or variables. For rules check in the rules option at the top.(Currently not available but will come soon)
  • LITERALS/CONSTANTS:- Those quantities which does not change their values during the program execution are called literals or constants.
  • PUNCTUATORS/SEPERATORS:- They are specific symbols used to indicate how group of codes are divided and arranged.
  • OPERATORS:- They are symbols to perform various operations with the help of operands.
